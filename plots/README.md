# Plot Status

Every PNG in this directory is a stale pre-audit artifact. The output hashes and
known generators are recorded in
`docs/provenance/legacy_artifact_manifest.csv`. These images include unsupported
`Ba_eff` bands, ambiguous expansion-ratio labels, in-sample prediction comparisons,
or fields whose measured/model provenance is unresolved.

Plot titles, captions, colors and apparent trends are not scientific findings.
The legacy plot generators are disabled until their inputs and labels have
complete provenance and each regenerated output receives a full manifest.
